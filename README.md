# Portfolio-Dashvsvs

Статичный сайт-портфолио для Даши Dashvsvs.

## Структура

- `index.html` - главная страница.
- `styles.css` - весь визуальный стиль сайта.
- `script.js` - меню и поведение шапки при прокрутке.
- `assets/` - изображения проекта.

## Как открыть

Открой `index.html` в браузере.

## Что заменить позже

- Тексты о Даше.
- Реальные работы и фотографии.
- Контакты в блоке `Contact`.
